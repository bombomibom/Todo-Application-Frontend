# Todo-Application-Frontend
React.js 사용 Todo 애플리케이션
React.js, 스프링 부트, AWS로 배우는 웹 개발 101 책으로 학습하는 프로젝트

## 프로젝트 소개
Todo 애플리케이션의 기본적인 기능을 구현한 후 인증 기능을 추가해 다수의 사용자를 지원.
<br>
